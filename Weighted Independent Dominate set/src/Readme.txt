--How to Compile
In this directory, enter the following command:
$ make
or
$ make all
It will generate the executable file " choose " 
If you want to remove it please enter the following command
$ make clean

--How to Run
Usage: ./<exe> <node_file_name> <edge_file_name> <output_file_name>
e.g.:
./choose ../benchmark/case1.nv ../benchmark/case1.as ../result/case.rpt
